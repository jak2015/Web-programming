Financial Calculator

This is a simple financial calculator, that can be used to calculate per month retirement savings, the amount you have invested and loan pay off. This can be very helful for planning the savings for retirement.

Retirement Amount - Savings Per Month
Parameter Name Description Example Value
yearsUntilRetirement This parameter shows years untill retirement. 10 years
amountSavingPerMonth This parameter shows amount to be saved per month. 200$
yearlyInterestRateOfInvestment This parameter shows yearly interest rate. 0.05
Invested Amount - After Some Years
Parameter Name Description Example Value
yearsInvesting This parameter shows years of investment. 5 years
initialAmount This parameter shows the initial amount of investment. 300$
yearlyInterestRateOfInvestment This parameter is used for Interest Rate of saving 0.05
Loan Payoff
Parameter Name Description Example value
monthlyPaymentAmount This shows monthly payment amount. 200$
initialLoanAmount This shows the initial investment amount. 1000$
yearlyInterestRateOfLoan This shows the yearly rate of interest. 0.05
